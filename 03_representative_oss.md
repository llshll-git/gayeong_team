# 3. 대표적인 오픈소스소프트웨어 소개

## 3.1 Linux
- 리누스 토발즈가 1991년 발표  
- 서버·클라우드·임베디드 등 광범위한 분야에서 활용  
- GPL 라이선스 기반

## 3.2 Visual Studio Code
- Microsoft가 개발한 오픈소스 코드 에디터  
- 확장성(Extension) 중심 구조  
- MIT 라이선스

## 3.3 TensorFlow
- Google이 공개한 딥러닝 프레임워크  
- 연구·서비스·모바일 환경까지 폭넓게 지원  
- Apache 2.0 라이선스
