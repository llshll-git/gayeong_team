01_OSS_Overview.md
오픈소스소프트웨어(OSS) 개요

작성자: 김OO(초안), 이OO(보완), 박OO(예시 추가)

오픈소스소프트웨어(Open Source Software)는 소스코드를 누구나 열람·수정·재배포할 수 있도록 공개한 소프트웨어를 의미한다.
초기에는 개발자 간 협력 목적이 강했으나, 현재는 기업·기관·개인 등 다양한 주체가 참여하는 글로벌 생태계로 성장하였다.

팀원 간 토론을 통해 OSS의 핵심 특징을 다음과 같이 정리했다.

소스코드 접근 가능

수정 및 재배포 허용

오픈소스 라이선스 준수 필요

공동 개발을 위한 협업 구조 발달

박OO 팀원이 실사용 예시를 추가해 전체 개념이 더 명확해졌다.

📄 02_OSS_History.md
오픈소스의 역사

작성자: 이OO(초안), 김OO(연표 재정리)

오픈소스의 역사는 자유 소프트웨어 운동(FSF)에서 시작된 것으로 알려져 있다.
리처드 스톨만이 주도한 GNU 프로젝트는 소프트웨어를 자유롭게 공유해야 한다는 철학을 강조했다.

이어진 팀 논의로 주요 흐름을 정리했다:

1983 – GNU 프로젝트 시작

1991 – 리누스 토발즈, 리눅스 공개

1998 – “Open Source” 용어 공식 사용

2000년대 – 대규모 오픈소스 생태계 형성

2010년대 이후 – GitHub 중심의 협업 문화 발달

마지막에 팀원 박OO가 깃허브가 오픈소스 확산의 핵심 플랫폼으로 자리 잡았다는 내용을 보완해 완성도를 높였다.

대표적인 오픈소스 사례

작성자: 박OO(초안), 이OO(정리), 김OO(설명 보충)

대표적인 OSS를 사례 중심으로 조사하였다.

1) Linux

전 세계 서버의 상당 부분이 사용

대규모 커뮤니티 기반 유지보수

2) Python

다양한 오픈소스 라이브러리와 생태계

AI, 데이터 분석, 웹 개발 등 폭넓은 분야 적용

3) Git

분산 버전관리 시스템

개발 협업 필수 도구로 자리 잡음

4) VS Code

마이크로소프트에서 공개한 편집기

확장 기능 중심의 오픈 생태계 구축
